# Required MVP Screenshots

Capture these screenshots for the MVP presentation package. Store image files in this directory using the suggested filenames.

## Required captures

1. `01-web-demo-home.png`
   - Public web demo loaded at https://project2-three-rho.vercel.app
   - Shows the main input/demo state before submitting a basket.

2. `02-web-demo-basket-result.png`
   - Web demo after submitting a representative basket.
   - Shows parsed items and the recommendation output.

3. `03-web-demo-mobile.png`
   - Mobile-width view of the web demo.
   - Confirms the MVP is presentable on a phone-sized viewport.

4. `04-api-docs-overview.png`
   - API docs loaded at https://macau-shopping-api.onrender.com/docs
   - Shows the FastAPI/OpenAPI endpoint list.

5. `05-api-docs-example.png`
   - Expanded API docs endpoint with request/response schema or example visible.

## Capture guidance

- Use a clean browser window without personal bookmarks or credentials visible.
- Include the browser address bar only when useful to prove the live deployed URL.
- Prefer 1440px desktop width for presentation screenshots.
- Prefer 390px to 430px viewport width for mobile screenshots.
- Do not include private `.env` values, deployment dashboards, tokens, or local file paths.
