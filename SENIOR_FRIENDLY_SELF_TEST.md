# Senior-friendly Self Test

不需要真實測試者，也可以先用以下方式自測。

## 測試原則

- 假設用戶不懂 product_oid。
- 假設用戶不懂 API / AI / signals。
- 假設用戶只想知道去哪買。
- 假設用戶不會閱讀太長說明。

## 測試任務

每項 0/1 分：

1. 30 秒內能否知道要按哪個按鈕？
2. 不看說明能否輸入購物清單？
3. 結果第一眼是否知道去哪間店？
4. 是否第一眼看到總價？
5. 是否知道為什麼推薦這間店？
6. 是否沒有出現英文技術字？
7. 手機寬度是否不爆版？
8. 如果 API 失敗，是否知道是系統問題而不是自己操作錯？
9. 是否能在不使用候選商品功能下完成任務？
10. 是否能在需要時找到「指定品牌 / 規格」？

## 評分

8 分以上才算可給 40+ 用戶試用。
